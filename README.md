# example-ag-grid-vue
Example Using ag-Grid and VUE taken from 
[ag-Grid Vue Example](https://www.ag-grid.com/vue-getting-started/)

## install ag-Grid 
Run this command to install ag-Grid:

```
npm install --save ag-grid-community ag-grid-vue

```
